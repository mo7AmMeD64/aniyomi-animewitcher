# AnimeWitcher — Aniyomi Extension

Extension لتطبيق Aniyomi يتصل بـ API الخاص بـ AnimeWitcher.

## التثبيت

1. اذهب إلى [Releases](../../releases)
2. حمّل آخر ملف `.apk`
3. افتح **Aniyomi** → Browse → Extensions
4. اضغط أيقونة الملف في الأعلى
5. اختر الـ APK

## المميزات

- تصفح كل الأنمي مع الصور
- بحث بالاسم عربي أو إنجليزي  
- جميع الحلقات مرتبة تحت كل أنمي
- تشغيل عبر Pixeldrain (سريع وثابت)
- اختيار الجودة من الإعدادات
